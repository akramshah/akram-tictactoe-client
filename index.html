About

This functional tic-tac-toe game was created by Akram Shah as a part of the SEI immersive at General Assembly. A user is able to create a username and password, log in, change password, see the number of games played, select X or O on the board, see a win, lose, or draw message, start a new game, and log out. This app was created using HTML, CSS, and Javascript. Please find the Github repo here and contact me here.
